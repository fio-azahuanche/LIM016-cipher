Instrucciones.

  1. Escriba el mensaje que quiera cifrar o descifrar.
  2. Ingrese el número de desplazamiento (es para indicar el nivel de seguridad).
  3. Presione el botón CIFRAR o DESCIFRAR.
  4. Copie su mensaje cifrado y listo para ser enviado.

NOTA: Si quiere descifrar su mensaje, deberá seleccionar el mismo número de desplazamiento con el que cifraste.
